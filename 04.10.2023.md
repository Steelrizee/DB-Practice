## Exercise 01 
```sql
SELECT id FROM menu 
EXCEPT
SELECT menu_id FROM person_order
ORDER BY id ASC;
```

## Exercise 02 

```sql
SELECT pizza_name, price, name FROM menu m
LEFT JOIN pizzeria ON m.pizzeria_id = pizzeria.id
WHERE m.id IN (SELECT id FROM menu 
EXCEPT
SELECT menu_id FROM person_order)
ORDER BY price ASC; 
```
